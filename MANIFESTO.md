# Calibrated Design

Calibrated Design is the philosophy. Restraint as differentiator.
"If it doesn't need to be there, it shouldn't be there."

## The four pillars

1. **Calibrated Design** — visual and interaction judgment
2. **Calibrated Voice** — copy and language judgment
3. **Calibrated AI** — model behavior and product judgment
4. **Calibrated Vibe Coding** — process and methodology judgment

Each pillar is the same discipline applied to a different surface:
tune for context, exercise taste, ship only what earns its place.

## Luxury psychology

Calibrated Design products feel premium not because they're loud or
feature-rich, but because every element earns its place. The user
feels someone *cared* — which most software fails to communicate.
Restraint is the differentiator.

## The two filmmakers

Tier 1: artisanal craft, slow and singular (e.g., Kubrick).
Tier 2: taste plus leverage — the auteur with a competent crew (most
great filmmakers).
Tier 3: AI slop — output without taste.

Calibrated Design is Tier 2. Taste directs; tools amplify; output
earns the user's attention.

## One signature move per project

Each Calibrated Design project commits to ONE signature interaction
or visual that distinguishes it from generic alternatives. Not three.
Not seven. One. Restraint enforces clarity; clarity enforces care.

Examples:
- Sumi world (STUND): viscous ink behavior
- Blueprint world (STUND): drafted-erasure feel
