# Super 100 — Dream List (Link-Building Foundation)

This is the curated list of ~100 relationship and link targets that every
Module 2 link-building tactic plugs into (Glen Allsopp SEO Blueprint 3,
Link Building Blueprint, L04/L06). The Super 100 is the strategy; the
individual tactics (outreach laws, angle plays, sourcing recipes) are just
different ways of sourcing and contacting the names on this list.

This file is a **reusable template**: the 13-category structure applies to
any property. The first instance below is populated for **Kinesthetic
Marketing**. Fill categories over time — some are researchable from outside
(publications, tools, communities); others require Director input (existing
relationships, partners, personal network) because only the Director has
that context.

## Appendix — GEO Citation Mechanics (site strategy, not dream-list)

- Reddit ≈ 46.7% of Perplexity citations; Wikipedia ≈ 47.9% of ChatGPT top citations — community + encyclopedic presence matters for citations.
- ChatGPT web search indexes via BING — submitting sitemap to Bing Webmaster Tools is a prerequisite for ChatGPT citation eligibility (free, ~10 min). Consider wiring IndexNow (ping-on-publish) as a follow-up — strengthens Bing + Brave-supplemental from one mechanism. Own ticket if pursued (needs an IndexNow key file at site root + ping on deploy).
- **Brave — no verification console exists (corrected 2026-06-30).** Brave has NO Google/Bing-style webmaster tool; its independent index is built via the opt-in Web Discovery Project (WDP) — real Brave users visiting the site. Cannot be submitted to directly; grows with traffic + inbound links. Two partial levers: (1) search.brave.com/webmasters MAY accept an XML sitemap submission to help BraveBot discover pages — UNVERIFIED/conflicting reports; worth a 2-min check, submit sitemap.xml if the endpoint is live, no-op if not. (2) IndexNow (Bing-supported protocol) — Brave uses Bing supplemental results where its index is thin, so IndexNow submission helps Bing + Brave-supplemental together = the real ownable lever. Why it matters: Claude web search ≈86.7% overlap with Brave's index; Brave also feeds Perplexity + Meta. Brave presence grows via the same link-building flywheel as Module 2 (more inbound links → more Brave/Claude citability).
- llms.txt at site root = emerging AI-crawler priority signal (Kinesthetic SHIPPED this 2026-06-30 — done).
- Typical 4–8 week lag from publish to first AI citation; ~6.8 day median to first citation per Profound data; uncited past ~37 days may signal a technical/robots block.
- GEO citation lift signals (peer-reviewed KDD 2024): +41% quotations, +32% statistics, +30% citations, +28% fluency.
