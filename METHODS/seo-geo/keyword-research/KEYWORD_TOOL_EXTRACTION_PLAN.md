# Keyword-Tool Extraction Battle-Plan (Module 2 + 3 tool-gated work)

This is the single executable checklist for every piece of SEO work that's
currently blocked on external keyword/backlink tooling — banked from the
Module 2 (Link Building) and Module 3 (Keyword Research) audits so the
Director can execute it in one disciplined pass rather than re-deriving
what's needed each time a tool gets pulled.

**Tool reality (2026, verified):** Ahrefs has **no free trial anymore** —
killed sitewide for trial-scraping abuse. There are three real access paths:

1. **Ahrefs Webmaster Tools (AWT)** — FREE forever, but **owned + verified
   sites only**, no competitor data.
2. **Ahrefs Starter** — $29/mo, monthly (cancel anytime), gives competitor
   data but **capped** (1,000 rows / 1,000 credits per month, no Content
   Explorer).
3. **SE Ranking or Semrush** — genuine 14-day free trial. SE Ranking
   includes AI-search tracking, which is on-brand for this business's GEO
   positioning.

**Execution principle:** do all FREE-tier extraction anytime, with no time
pressure. Batch all PAID-tier extraction into **one disciplined blitz
window**. Export everything to CSV as you go — dashboards lock the moment
access ends or a trial expires; exports are yours forever. Treat this as a
data-harvesting operation, not a browsing session.

## Section 1 — Free tier (Ahrefs Webmaster Tools, $0, do anytime)

_Owned + verified properties only. No competitor data. No time pressure._

This document's scope is **kinestheticmarketing.com**. Portfolio note: the
Director may also want to verify other owned properties (DCI /
denvercoloradoinsulation.com, Don's Garage, ToteFinders) in AWT for their
own free baseline data — flagged here, not detailed, since this plan is
Kinesthetic-specific.

- [ ] Verify kinestheticmarketing.com in Ahrefs Webmaster Tools (GSC or DNS verification).
- [ ] Export: own ranking keywords — which queries the site already ranks for + current positions. Cross-reference against GSC's own data. Feeds Module 3 L05 (the $50k keyphrase) and L08 (best-company qualification queries).
- [ ] Export: own backlink profile — referring domains, anchor text, new/lost links. This is the baseline for Module 2 — know your own links before hunting competitors'. Feeds Super 100 Category 10.
- [ ] Run Site Audit: technical issues (broken links, redirects, missing meta, slow pages). Feeds any future on-site cleanup pass.
- [ ] Note what AWT **cannot** give you: competitor data, keyword search volumes at scale, Content Explorer. Those all live in Section 2.

## Section 2 — Paid blitz (Ahrefs Starter $29 OR SE Ranking/Semrush 14-day trial)

_Competitor + market data. Time/credit-limited. Execute as one focused blitz. Export everything to CSV._

### Module 3 (keyword research) paid items

- [ ] **L06 — title-word-cloud mining.** Pull the competitor title-tag corpus (Ahrefs Site Explorer / Content Explorer, or the SE Ranking equivalent) and extract recurring money-words.
- [ ] **L06 — buried-page discovery.** `site:` + date-range search, or a top-pages export on competitors, to find their hidden winners.
- [ ] **L04 — SERP-feature filter.** Find which target queries currently show a *winnable* featured snippet. The atomic-answer pattern is already built on 11 pages — this tells you where to point it.
- [ ] **L13 — "[thing] PDF" search demand.** Keyword-ideas + include-filter for PDF-format demand, to decide which educational page gets the gated-PDF treatment.
- [ ] **L05 / L08 — volume + difficulty for the Denver ranking cluster** ("ai seo denver" etc.) and qualification queries ("best/reviews/legit/worth it") — sharpens the page already ranking ~#71.
- [ ] **L11 (gated further)** — fast-riser reverse-engineering needs DataForSEO/licensed SERP data, **not** standard Ahrefs/SE Ranking. Flag as a separate spend decision — likely skip for this blitz.

### Module 2 (link-building) paid items

- [ ] **L02 — competitor backlink export.** Pull the referring domains of agencies ranking for your target terms → feeds Super 100 Category 10 + outreach targets. (Also partially answerable free via detailed.com/links.)
- [ ] **L25 — fast-linkers to recent content.** Content Explorer (Ahrefs Standard+ only — this may exceed the $29 Starter tier; note the gap before relying on it) → who links quickly to new content in this niche.
- [ ] **L18 (TLB-24 half) — "has anything already snowballed a link?"** Check current backlink state on the resources pages. Partially answerable free via AWT on the own-site export in Section 1 — only pull this in the paid blitz if AWT's data is insufficient.

## Section 3 — Execution notes

- **Order:** own-site free extraction first (Section 1) — it sets the
  baseline. Then the paid competitor blitz (Section 2), so you know your
  own gaps before spending credits/trial-days on competitor data.
- **Export format:** one CSV per checklist item, dropped into a dated
  folder. These exports are the working dataset for weeks after tool
  access ends — the blitz window is short, the data outlives it.
- **Tool choice:** if the priority is competitor backlinks + word-cloud
  mining, Ahrefs $29 Starter is the cheapest way to get Ahrefs-specific
  data — but respect the 1,000-row cap and prioritize ruthlessly. If the
  priority is a broader 14-day window that also covers AI-search tracking,
  use the SE Ranking free trial instead.
- **Items that need NONE of this tooling** — do these independently of the
  blitz, anytime, for free:
  - L01 / L07 — canon-rule adoptions (keyword-selection lens, stick-to-the-wall testing) — process changes, not data pulls.
  - L10 — Quora Ads Manager signup (free).
  - L09 — manual competitor SERP spot-checks (free, just time).
  - L08 — the core qualification-query thinking exercise, runnable today from existing GSC data alone.
